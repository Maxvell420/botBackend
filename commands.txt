для создания виртуального окружения
apt install python3.10-venv
создание виртуального окружения в текущей папке где name имя
python3 -m venv .venv

Для понятия отключено ли окружение можно посмотреть в терминале есть ли в круглых скобках имя окружения
активация окружения
source .venv/bin/activate
Отключение
deactivate 

добавление репозитория в гит:
git remote add origin git@github.com:Maxvell420/botBackend.git

запуск сервера
uvicorn app.main:app --reload

сохранение библиотек различных подгружаемых
pip freeze > requirements.txt

установка из окружения нужных библиотек
pip install -r requirements.txt

добавить содержимое .venv в гитигнор
echo "*" > .venv/.gitignore

зайти в докер контейнер например с именем python
docker exec -it python bash

Для того чтобы работали подсказки из IDE нужно поднимать виртуальное окружение и загружать в него библиотеки, 
затем уже можно подгружать все что нужно (иногда в vscode нужно переключать интерпретатор)

Установка расширений для убунты .deb
sudo dpkg -i $path

Зайти в mysql
mysql -u root -p
получить всех юзеров
SELECT User, Host FROM mysql.user;
